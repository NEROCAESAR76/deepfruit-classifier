# DeepFruit Classifier 🍎🍌🍍

DeepFruit Classifier is a deep learning project for multi-class fruit image classification using Convolutional Neural Networks (CNN) and TensorFlow.

## Features
- Multi-class fruit classification
- CNN Sequential architecture
- TensorFlow & Keras implementation
- TensorFlow Lite conversion
- TensorFlow.js conversion
- Model inference support

## Dataset
This project uses the Fruits-360 dataset.

## Classes
- Apple Red 1
- Banana 1
- Orange 1
- Strawberry 1
- Pineapple 1

## Model Architecture
- Conv2D
- MaxPooling2D
- Dense Layer
- Dropout
- Softmax Classification

## Technologies
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Results
The model achieved high accuracy on training, validation, and testing datasets.
